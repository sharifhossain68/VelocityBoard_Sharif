
## VelocityBoard Management System

## Overview
A full-stack ASP.NET Core 8 application for managing projects and tasks.

## Tech Stack
- ASP.NET Core 8
- Razor Pages + Web API
- Entity Framework Core
- SQL Server
- JWT Authentication
- Bootstrap

## Features
- User Authentication
- Task Add & Edit
- Dashboard 
- Responsive UI

## Setup
1. Clone repository
2. Update connection string
3. Run `update-database`
4. Run API & Web projects
